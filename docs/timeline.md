# Timeline

OpsFlow Lab is planned around Prince's expected build window of 2-3 hours per day.

## Planning Assumption

12-15 productive hours per week.

This accounts for implementation, debugging, visual review, documentation, Git commits, and handoffs.

## Official Timeline

- 6-week MVP
- 12-week Portfolio v1
- 16-week Portfolio v2 stretch
